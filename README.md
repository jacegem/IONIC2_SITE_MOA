# IONIC2_SITE_MOA

## 설치
```sh
npm install -g cordova ionic@beta
```

## 시작
```sh
ionic start sitemoa sidemenu --v2
```

## Plugin

### admobpro
```sh
ionic plugin add cordova-plugin-admobpro
```

### inappbrowser
```sh
ionic plugin add cordova-plugin-inappbrowser
```

### whitelist
```sh
ionic plugin add cordova-plugin-whitelist
```
